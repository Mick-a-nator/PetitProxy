Petit Proxy

A light-weight MTG proxy generator for rapid deck prototyping.

About

Proxys are for testing how a deck works, they don't have to be pretty. Save paper and ink by printing only the relevant information.

These proxys are small, so depending on your printer margins you should be able to print 12-20 cards on a single A4 page. To make best use of these proxies, slot them into the front of a sleeved basic land, preferably not of a colour used in the deck.

Enter your decklist above, one card per line, with the number of copies at the start (e.g. 4 Ancestral Recall). The form should accept major plain-text decklist formats ("4x", "4 - " "4" are all accepted ways of formatting multiples) and will attempt to make an exact match for a card name by default. If you choose to disable this behaviour, you can still try an exact match for a single card name by wrapping it in quotes (e.g. 2 "Mox Ruby").

Entering one side of a double-faced card will add both sides. This applies to split cards too (e.g. Searching for "Turn" will add "Turn" and "Burn" as 2 proxies) but currently this is not supported for Meld cards.
